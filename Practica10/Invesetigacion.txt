canRead().- Comprueba si la aplicación puede leer el archivo indicado por esta ruta abstracta.

canWrite().- Comprueba si la aplicación puede modificar el archivo indicado por esta ruta abstracta.

createNewFile().- Crea atómicamente un nuevo archivo vacío con el nombre de esta ruta abstracta si y sólo si aún no existe un archivo con este nombre.

getName().- Devuelve el nombre del archivo o directorio indicado por esta ruta abstracta.

getPath().- Convierte esta ruta abstracta en una cadena de ruta.

length().- Devuelve la longitud del archivo indicado por esta ruta abstracta.

isDirectorio() 	
Comprueba si el archivo indicado por esta ruta abstracta es un directorio.

mkdir() 	
Crea el directorio nombrado por esta ruta abstracta.

mkdirs() 	
Crea el directorio nombrado por esta ruta abstracta, incluyendo cualquier directorio padre necesario pero inexistente.

 	getArchivoPadre() 	
Devuelve la ruta abstracta del directorio padre de este nombre de ruta abstracto, o null si este nombre de ruta no nombra ningún directorio padre.

createTempFile(Cadena prefijo, Cadena sufijo) 	
Crea un archivo vacío en el directorio de archivos temporales por defecto, utilizando el prefijo y sufijo dados para generar su nombre.

 	createTempFile(Prefijo de cadena, Sufijo de cadena, Directorio de archivos) 	
Crea un nuevo archivo vacío en el directorio especificado, utilizando las cadenas de prefijo y sufijo dadas para generar su nombre.
